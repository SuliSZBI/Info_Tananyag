A Java nyelvcsalád

A Java nyelvcsalád elemei alatt azokat a nyelveket, technológiákat és eszközöket értjük, amelyek a Java programozási nyelvből származnak, arra épülnek, vagy szorosan együttműködnek vele a Java ökoszisztémán belül.

A Java nyelvcsalád főbb elemei:

Java SE (Standard Edition)
  • Az alap Java API és nyelvi eszköztár.
  • Tartalmazza a JVM-et (Java Virtual Machine) és a JDK-t (Java Development Kit).

Forrás: Oracle

Java EE / Jakarta EE (Enterprise Edition)
  • Vállalati szintű alkalmazásfejlesztéshez.
  • Tartalmaz webes, adatbázis- és tranzakciókezelő technológiákat (pl. Servlets, JSP, EJB, JPA).

Forrás: Oracle

Forrás: Jakarta

Java ME (Micro Edition)
  • Korlátozott erőforrású eszközökre (pl. beágyazott rendszerek, mobiltelefonok).

Forrás: Oracle

GraalVM
  • Több nyelvet támogató, Java-kompatibilis virtuális gép (JVM alternatíva), natív képességekkel.

Forrás: GraalVM

Kotlin
  • Teljesen interoperábilis a Javával.
  • A JetBrains fejlesztette, hivatalos Android-nyelvként támogatott.

Forrás: Kotlin

Scala
  • Funkcionális és objektum-orientált elemeket vegyít.
  • JVM-en fut, erősen típusos.

Forrás: Scala

Groovy
  • Dinamikus nyelv, egyszerűbb szintaxissal, gyakori a build (pl. Gradle) és scripting környezetekben.

Forrás: Groovy

Clojure
  • Funkcionális, Lisp-alapú nyelv a JVM-re.

Forrás: Clojure

Jython
  • Python nyelvű programokat futtat a JVM-en.

Forrás: Jython

JRuby
  • Ruby nyelv JVM-es implementációja.

Forrás: JRuby

Apache Ant / Maven / Gradle
  • Java-projektek építését, menedzselését támogató eszközök.

Forrás: Gradle
