A .NET keretrendszer (framework)

A .NET keretrendszer (vagy .NET platform) a Microsoft által fejlesztett, nyílt forráskódú fejlesztési környezet és futtatókörnyezet, amely lehetővé teszi különböző típusú alkalmazások (például asztali, webes, mobil, játék, felhő) fejlesztését és futtatását több nyelven (pl. C#, F#, VB.NET).

A legfontosabb célja az, hogy egységes környezetet biztosítson a fejlesztéshez és a futtatáshoz, függetlenül attól, hogy milyen eszközön vagy operációs rendszeren fut az adott alkalmazás. Két fő részből áll:

  • .NET futtatókörnyezet (Runtime) - például a Common Language Runtime (CLR) vagy a CoreCLR, amely kezeli a memóriát, hibákat, biztonságot stb.
  • .NET osztálykönyvtárak (FCL - Framework Class Library) - előre megírt, újrafelhasználható kódmodulok például fájlkezeléshez, hálózati kommunikációhoz, adatbázis-kezeléshez, stb.

Mire használható a .NET keretrendszer?

Alkalmazástípus, terület Példa, leírás
Asztali alkalmazások Windows Forms, WPF (Windows Presentation Foundation), MAUI (cross-platform)
Webalkalmazások és API-k ASP.NET, ASP.NET Core, Blazor
Mobilalkalmazások .NET MAUI, Xamarin (Android, iOS)
Felhőszolgáltatások Azure-integráció, microservice-ek
Játékfejlesztés Unity (C# alapú, .NET-tel működik)
IoT és beágyazott rendszerek .NET nanoFramework, Meadow F7, .NET IoT csomagok és eszközök
Konzolos (parancssori) eszközök, szkriptek Automatizálás, rendszerfelügyelet, gyors CLI-alkalmazások készítése

Főbb elemei a .NET ökoszisztémának:

Elem Szerep
.NET SDK A fejlesztéshez szükséges fordítók, eszközök.
.NET Runtime A programok futtatásához szükséges környezet.
C# nyelv A legnépszerűbb nyelv a .NET környezetben.
BCL (Base Class Library) Alapvető osztályok (fájlkezelés, hálózat, kollekciók stb.).
NuGet Hivatalos .NET csomagkezelő: könyvtárak letöltése.
MSBuild A projektek építésének rendszere.
Visual Studio / VS Code Fejlesztőkörnyezetek (IDE-k).

A .NET család főbb változatai:

Platform Rövid leírás
.NET Framework 2002-től, csak Windows-on, pl. WinForms, WPF.
.NET Core Modern, gyorsabb, cross-platform (2016-2020).
.NET 5+ / .NET 6, 7, 8… 2020-tól: .NET Core és .NET Framework összeolvasztva egy új platformba.

Jelenleg a hivatalos és aktívan fejlesztett verziók a .NET 6, .NET 7, .NET 8 (2024-től LTS támogatással), stb.

Miért használd a .NET-et?

  • Keresztplatformos: Windows, Linux, macOS.
  • Stabil és jól támogatott.
  • Magas szintű biztonság és memória-kezelés.
  • Magas szintű nyelvi eszközök (pl. async/await, LINQ).
  • Gazdag könyvtári ökoszisztéma.
  • Kiváló IDE támogatás (Visual Studio, Rider, VS Code).
  • Nyílt forráskódú (GitHub-on elérhető).
  • Nagy és aktív fejlesztői közösség.
  • Teljes Azure-felhő integráció.

Mikor érdemes használni a .NET-et?

  • Ha C# nyelven szeretnél fejleszteni.
  • Ha modern webes vagy mobilalkalmazást építesz.
  • Ha Windows alkalmazást vagy játékot készítesz Unity-ben.
  • Ha Microsoft-technológiákhoz szeretnél szorosan illeszkedni (pl. SQL Server, Azure, Office).
