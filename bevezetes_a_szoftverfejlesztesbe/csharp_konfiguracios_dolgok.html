A C# fejlesztési ökoszisztéma

A C# nyelv a .NET ökoszisztéma része, és a fejlesztés során többféle forrás-, konfigurációs fájl, futtatókörnyezet és keretrendszer (framework) játszik szerepet.

Forrásfájlok a C# nyelvben:

A forrásfájlok tartalmazzák a program logikáját, azaz a fejlesztő által írt kódot, amit a fordítóprogram lefordít és futtatható alkalmazássá alakít.

  • A program viselkedésének, működésének meghatározása.
  • Modulok, osztályok, függvények deklarálása.
  • Logikai folyamatok vezérlése.
Fájltípus Leírás
.cs C# forráskód fájl. Osztályokat, interfészeket, metódusokat tartalmaz.
.csproj Projektfájl, XML-formátumban - tartalmazza a fordítási beállításokat, fájlokat, függőségeket.
.sln Megoldásfájl (solution), Visual Studio projekteket egyesít.
.dll Dinamikusan betölthető .NET assembly - gyakran projekt eredménye.
.exe Futtatható fájl, ha a projekt típusától függően azt hoz létre.

Konfigurációs fájlok a C# nyelvben:

A konfigurációs fájlok a program futtatásának környezeti és működési beállításait tartalmazzák - gyakran különállóan a forráskódtól.

  • Külső beállítások megadása (pl. adatbázis elérés, API-kulcsok, fájlútvonalak).
  • Futtatási környezet beállítása (pl. fejlesztői vagy éles környezet).
  • Fordítási paraméterek (pl. mely fájlokat fordítsa le, milyen verziójú SDK-t használjon).
  • Projektstruktúra és függőségek leírása.
Előny Magyarázat
Kód és konfiguráció szétválasztása. Könnyebb karbantartás, újrakonfigurálás újrafordítás nélkül.
Környezetspecifikus működés. Ugyanaz a kód másként viselkedhet dev/test/prod környezetekben.
Automatizálás támogatása. Könnyebb CI/CD folyamatok kialakítása.
Fájl Leírás
app.config / web.config Klasszikus XML-alapú konfigurációs fájlok .NET Framework alatt.
appsettings.json Modern JSON-alapú konfiguráció .NET Core és .NET 5+ alatt.
launchSettings.json A fejlesztési környezet futtatási beállításait tartalmazza.
global.json .NET SDK verziót határoz meg több projekt esetén.
Directory.Build.props / targets MSBuild közös konfigurációk.

C# futtatókörnyezetek:

Név Funkció
.NET Framework Klasszikus, Windows-only platform (pl. 4.8).
.NET Core Cross-platform futtatókörnyezet (2016-2020).
.NET 5/6/7/8 Egyesített, modern platform (a .NET Core utódja).
Mono Nyílt forráskódú .NET implementáció (hasznos mobil és Unity fejlesztésben).
Blazor WebAssembly Runtime Böngészőben futó .NET környezet (C# WebAssembly-re fordítva).

Népszerű C# frameworkök:

Név Leírás
ASP.NET Core Modern, platformfüggetlen webes és API fejlesztési keretrendszer.
Blazor Webes frontend fejlesztés C# nyelven, WebAssembly vagy szerveres rendereléssel.
SignalR Realtime kommunikáció webalkalmazásokhoz.

Asztali alkalmazások:

Név Leírás
