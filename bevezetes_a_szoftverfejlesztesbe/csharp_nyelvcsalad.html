A C# nyelvcsalád

A C# nyelvcsalád alatt általában nem egy hivatalos nyelvcsaládot értünk (mint például a C nyelveknél - C, C++, Objective-C stb.), hanem azokat a nyelveket, technológiákat és eszközöket, amelyek szorosan kapcsolódnak a C# nyelvhez, főként a .NET ökoszisztéma részeként. Ezeket az alábbi kategóriák szerint lehet csoportosítani.

C# és kapcsolódó nyelvek:

Ezek olyan nyelvek, amelyeket a .NET platformra fejlesztettek, és hasonló célokat szolgálnak.

Nyelv Leírás
C# Fő .NET nyelv, általános célú, objektumorientált és modern.
Visual Basic .NET (VB.NET) Egyszerűbb szintaxisú .NET nyelv, főként üzleti alkalmazásokhoz.
F# Funkcionális nyelv a .NET keretrendszerben, erősen típusos.
JScript.NET A JavaScript .NET változata (ma már elavult).
PowerShell .NET-alapú szkriptnyelv, rendszeradminisztrációhoz.

C#-hoz szorosan kapcsolódó technológiák:

Technológia Kapcsolódás
.NET (Core, 5/6/7+) A C# nyelv futtatókörnyezete.
ASP.NET / ASP.NET Core Webes fejlesztés C#-ban
Entity Framework ORM (objektum-relációs leképezés) C#-hoz.
Xamarin / MAUI Mobil- és multiplatform fejlesztés C#-ban.
Blazor WebAssembly-alapú frontend fejlesztés C#-pal.
Roslyn A C# fordítóplatformja (compiler-as-a-service).

IDE-k és eszközök:

Eszköz Leírás
Visual Studio Teljes értékű fejlesztőkörnyezet C#-hoz.
Rider (JetBrains) Alternatív IDE C# fejlesztéshez.
Visual Studio Code Könnyedebb, bővíthető szerkesztő C#-hoz is (OmniSharp kiegészítővel).
