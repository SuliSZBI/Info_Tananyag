A Java fejlesztési ökoszisztéma

A Java fejlesztési ökoszisztémájában kulcsfontosságúak a forrásfájlok, konfigurációs fájlok, a futtatókörnyezetek és a különféle keretrendszerek (frameworkök).

Forrásfájlok a Java nyelvben:

A forrásfájlok tartalmazzák a program logikáját, azaz a fejlesztő által írt kódot, amit a fordítóprogram lefordít és futtatható alkalmazássá alakít.

  • A program viselkedésének, működésének meghatározása.
  • Modulok, osztályok, függvények deklarálása.
  • Logikai folyamatok vezérlése.
Fájltípus Leírás
.java Java nyelven írt forráskód.
.class Fordított Java bytecode (a .java fájlokból a javac parancs készíti).
.jar Java archívum - tömörített fájlok, amely több .class, konfiguráció, metaadatot tartalmaz.

Konfigurációs fájlok a Java nyelvben:

A konfigurációs fájlok a program futtatásának környezeti és működési beállításait tartalmazzák - gyakran különállóan a forráskódtól.

  • Külső beállítások megadása (pl. adatbázis elérés, API-kulcsok, fájlútvonalak).
  • Futtatási környezet beállítása (pl. fejlesztői vagy éles környezet).
  • Fordítási paraméterek (pl. mely fájlokat fordítsa le, milyen verziójú SDK-t használjon).
  • Projektstruktúra és függőségek leírása.
Előny Magyarázat
Kód és konfiguráció szétválasztása. Könnyebb karbantartás, újrakonfigurálás újrafordítás nélkül.
Környezetspecifikus működés. Ugyanaz a kód másként viselkedhet dev/test/prod környezetekben.
Automatizálás támogatása. Könnyebb CI/CD folyamatok kialakítása.
Fájltípus / technológia Leírás
manifest.mf JAR fájlok metaadatait tartalmazza.
application.properties / application.yml Spring Boot konfigurációs fájl.
pom.xml Maven build-eszköz konfigurációs fájlja (projektfüggőségek, pluginok).
build.gradle Gradle build-eszköz konfigurációs fájlja (alternatíva a Mavenhez).
web.xml Régebbi Java EE webes alkalmazások konfigurációs fájlja.
log4j.properties / logback.xml Naplózási konfigurációk (Log4j, Logback stb.).

Java futtatókörnyezetek:

Rövidítés Név Funkció
JDK Java Development Kit Fejlesztéshez: tartalmazza a fordítót (javac), debugger-t, eszközöket.
JRE Java Runtime Environment Csak a futtatáshoz szükséges: JVM + alap könyvtárak.
JVM Java Virtual Machine A bytecode végrehajtásáért felelős virtuális gép.
OpenJDK Nyílt forrású Java fejlesztői környezet.
GraalVM Alternatív JVM, több nyelv támogatásával és natív képességekkel.

Népszerű Java frameworkök:

Név Leírás
Spring / Spring Boot Moduláris, könnyű webes és vállalati fejlesztéshez.
Jakarta EE (korábban Java EE) Oracle/Apache által karbantartott vállalati alkalmazás platform.
JSF (JavaServer Faces) Komponens-alapú webes UI keretrendszer.
Vaadin Java alapú frontend framework modern UI-val.

ORM és adatkezelés:

Név Leírás
Hibernate Objektum-relációs leképezés (ORM) keretrendszer.
JPA Java Persistence API - ORM szabvány, Hibernate az egyik implementáció.
MyBatis Kézi SQL-t használó adatlekérdezési keretrendszer.
