(Asztali alkalmazások fejlesztése - Bevezetés a szoftverfejlesztésbe)

Népszerű C# fejlesztőkörnyezetek (IDE-k)

A C# fejlesztőkörnyezetek megismerése alapvető lépés a .NET-alapú alkalmazások készítéséhez. A Microsoft által fejlesztett C# nyelvet elsősorban a .NET ökoszisztémában használják, így a fejlesztőkörnyezetek szorosan integráltak ehhez a platformhoz.

Alapfogalmak

Fogalom Leírás
.NET SDK (Software Development Kit) A C# alkalmazások fejlesztéséhez szükséges eszköztár (fordító, könyvtárak, CLI eszközök).
.NET Runtime A C# alkalmazások futtatásához szükséges környezet.
IDE (Integrated Development Environment) Olyan alkalmazás, amely integráltan támogatja a fejlesztést (kódszerkesztés, fordítás, hibakeresés, stb.).

Fejlesztőkörnyezetek

  1. Visual Studio (Windows)
    • Teljes körű IDE C#, .NET, ASP.NET, WPF, MAUI, Blazor stb. támogatással.
    • Professzionális eszközök hibakereséshez, refaktoráláshoz, teszteléshez.
    • Verziók: Community (ingyenes), Professional, Enterprise.
    Forrás: Visual Studio
  2. Visual Studio Code (Windows, macOS, Linux)
    • Könnyű, gyors szerkesztő bővítményekkel.
    • A C# for Visual Studio Code (Omnisharp) bővítmény segítségével támogatja a C#-ot.
    • Jó választás kisebb projektekhez, cross-platform fejlesztéshez.
    Forrás: Visual Studio Code
  3. JetBrains Rider
    • Keresztplatformos (cross-platform), gyors IDE C#-hoz, .NET Core-hoz, Unity-hoz.
    • IntelliJ-alapú, erős refaktorálási és hibakeresési lehetőségek.
    • Kereskedelmi licenc szükséges.
    Forrás: JetBrains Rider

C# fejlesztési eszközök és CLI

Eszköz Leírás
.NET CLI Parancssori eszköz projektek létrehozására, buildelésre, futtatásra (dotnet new, dotnet build, stb.).
NuGet A .NET csomagkezelője: könyvtárak telepítése, kezelés.
MSBuild A .NET projektek építési rendszere, .csproj fájlokat dolgoz fel.

C# fejlesztésre alkalmas platformok

Platform Támogatás
Windows Teljes körű támogatás Visual Studio-val.
Linux .NET SDK + VS Code / Rider.
macOS .NET SDK + VS Code / Rider / Visual Studio for Mac (utóbbi megszűnik 2024 után).
